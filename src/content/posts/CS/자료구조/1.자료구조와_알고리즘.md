# 1.자료구조와 알고리즘

알고리즘: 컴퓨터로 문제를 풀기 위한 단계적 절차

기술 방법

- 흐름도
- 유사 코드
- 프로그래밍 언어
- 자연어

자료형: 데이터의 종류

추상 데이터 타입

- 데이터 타입을 추상적으로 정의한 것
- 데이터나 연산이 무엇인가는 정의되지만 데이터나 연산을 어떻게 컴퓨터 상에서 구현할 것인지는 정의되지 않는다.
- 추상화 -> 정보은닉기법 -> 추상 자료형(ADT)
- 추상화란 사용자에게 중요한 정보는 강조, 중요하지 않은 세부사항 제거

객체: 추상 데이터 타입에 속하는 객체

연산: 객체들 사이의 연산.

O()

복잡도 분석의 종류

- 시간복잡도
- 공간복잡도

O(1) - O(log(n)) - O(n) - O(nlogn) - O(n^2) - O(n^3) - O(2^n) - O(n!)

빅 오메가, 빅 세타 등…

최선,평균,최악의 경우

ex) 순차탐색

- 최선: 찾고자 하는 숫자가 맨 앞 O(1)
- 최악: 맨 뒤 O(n)
- 평균: 중간 O(n)

최악의 경우가 가장 널리 사용됨.

최악이 널리 사용되니 O(n)
