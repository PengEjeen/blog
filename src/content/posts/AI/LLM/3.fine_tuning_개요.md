# Fine tuning 개요

참고 - [https://seungbeomdo.tistory.com/79](https://seungbeomdo.tistory.com/79)

> Fine tuning이란 pre-trained 모델을 **구체적인 태스크에 맞게 추가 훈련**하는 과정.

## 1. Fine tuning 방법

- **end-to-end** — 주어진 pre-trained 모델의 **모든 레이어**를 추가 훈련
- **Freezing** — 일부 레이어는 그대로 두고 일부 레이어만 추가 훈련

### 1.1 Freezing 방법

- 뉴럴넷의 backend 레이어들을 일종의 **특성 추출기**처럼 사용
- 추출된 특성을 인풋으로 하는 별도의 뉴럴넷을 만들어 훈련
- 결과적으로 backend 레이어는 고정된 상태에서 frontend 레이어만 추가 훈련

### 1.1.1 Tokenizer

- pytorch pipeline 함수만으로 구현하기 어려움. 모델을 분해하는 과정이 필요하기 때문.
- 사용 모델: `facebook/bart-large-mnli`
