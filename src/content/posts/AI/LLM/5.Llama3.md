# Llama3 (with NVIDIA TensorRT-LLM)

튜토리얼 - [NVIDIA Developer Blog: Turbocharging Meta Llama 3 Performance with TensorRT-LLM and Triton](https://developer.nvidia.com/ko-kr/blog/turbocharging-meta-llama-3-performance-with-nvidia-tensorrt-llm-and-nvidia-triton-inference-server/)

NVIDIA TensorRT-LLM의 Meta Llama3 추론 최적화 메모.

## 1. 설치

Docker 컨테이너 내부에 TensorRT-LLM 설치에 필요한 모든 종속성 설치.

```bash
git clone -b v0.8.0 https://github.com/NVIDIA/TensorRT-LLM.git
cd TensorRT-LLM
```

## 2. 모델 가중치 가져오기

Hugging Face Hub에서 Llama3-8b 모델의 instruction-tuned 변형의 가중치를 가져옴.

```bash
git lfs install
git clone https://huggingface.co/meta-llama/Meta-Llama-3-8B-Instruct
```

## 3. TensorRT-LLM 컨테이너 실행

기본 Docker 컨테이너 시작 + TensorRT-LLM에 필요한 종속성 설치.

```bash
# 기본 docker 이미지 환경 시작
docker run --rm --runtime=nvidia --gpus all \
  --volume ${PWD}:/TensorRT-LLM \
  --entrypoint /bin/bash -it \
  --workdir /TensorRT-LLM \
  nvidia/cuda:12.1.0-devel-ubuntu22.04

# 종속성 설치 (TensorRT-LLM은 Python 3.10 요구)
apt-get update && apt-get -y install python3.10 python3-pip openmpi-bin libopenmpi-dev

# stable 버전 설치 (clone한 브랜치에 대응)
pip3 install tensorrt_llm==0.8.0 -U --extra-index-url https://pypi.nvidia.com
```

## 4. 모델 컴파일

모델 가중치 + TensorRT-LLM Python API로 작성된 모델 정의 → TensorRT 엔진으로 컴파일.

```bash
# huggingface-cli 로그인 (토큰: huggingface.co/settings/token)
huggingface-cli login --token *****

# Llama 8B 모델 single GPU + BF16 빌드
python3 examples/llama/convert_checkpoint.py \
  --model_dir ./Meta-Llama-3-8B-Instruct \
  --output_dir ./tllm_checkpoint_1gpu_bf16 \
  --dtype bfloat16

trtllm-build \
  --checkpoint_dir ./tllm_checkpoint_1gpu_bf16 \
  --output_dir ./tmp/llama/8B/trt_engines/bf16/1-gpu \
  --gpt_attention_plugin bfloat16 \
  --gemm_plugin bfloat16
```

> TensorRT-LLM API로 모델 정의를 생성할 때 뉴럴 네트워크의 레이어를 형성하는 TensorRT primitive에서 연산 그래프 구축. 이러한 작업은 GPU용으로 사전 작성된 프로그램인 특정 커널에 매핑된다.
