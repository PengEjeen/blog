# RNN(Recurrent Neural Network) 개요

참고 - [https://brightwon.tistory.com/10](https://brightwon.tistory.com/10)

## 1. 개요

- 연속된 데이터, 즉 순차 데이터 사용
- 시계열 특성을 지닌 데이터 학습

![RNN 구조](./assets/2-1.svg)

- 이전 상태로부터 전달된 값을 **히든 스테이트**라고 함. → 이는 현재의 영향을 받아 매번 갱신
- 현재 타임 스텝의 **히든 스테이트 ht**는 **인풋 Xt**와 **이전 히든 스테이트 ht-1**의 영향을 받아 갱신된다.

## 2. 학습 방법

- 경사하강법, 오차 역전파
- 시계열 특성이 있기 때문에 **BPTT(Back-Propagation Through Time)** 사용

## 3. 한계와 보완

- 역전파 도중, 과거로 올라갈수록 **gradient 소실 문제** 발생
- RNN이 처리하지 못하는 **장기 의존성 문제**를 다루기 위해 **LSTM** 고안
- gradient 소실을 예방하기 위해 활성화 함수로 **tanh** 사용
