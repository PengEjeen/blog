# Snapgoal Project (개요 / 인덱스)

축구 영상에서 특정 인물·이벤트를 자동으로 추적·검출해 하이라이트를 만드는 프로젝트.

## 진행 단계

1. **기술조사** — Detection / Tracking / Re-ID 모델 후보 비교 → `기타/프로젝트/1.Snapgoal_프로젝트_기술조사.md`
   - 탐지: YOLOv3/v5/v6/X, MMDetection, PP-YOLOE, Mask R-CNN, RT-DETR
   - 추적: DeepSORT, ByteTrack, BoT-SORT, StrongSORT
   - Re-ID: FastReID
   - 액션 인식: MMAction2, ActionSpotting (NetVLAD++, SoccerNet)
   - 최종 파이프라인 후보: **YOLOX + BoT-SORT + FastReID**

2. **검증 / 실험**
   - 객체 겹침(occlusion) 완화
   - ID switching 문제 해결
   - 30초 클립 처리 시간 ≈ 8m 32s, VRAM 2GB (g4dn.xlarge)

3. **MVP 구현 (FastAPI 기반 서비스)**
   - `GET /` — 클라이언트(index.html)
   - `GET /tracking_feed` — MJPEG 스트리밍
   - `POST /set_roi` — ROI 지정
   - `POST /pause`, `/resume`, `/stop`

## 핵심 트레이드오프

| 우선순위 | 추천 조합 |
|---------|----------|
| 속도 우선 | YOLOv6-n + Deep SORT |
| 속도/정확도 밸런스 | YOLOX-S + ByteTrack |
| 정확도 우선 | PP-YOLOE-S + BoT-SORT |
