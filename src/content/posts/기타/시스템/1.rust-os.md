# rust-os (Rust 기반 OS 만들기)

참고 - [http://rust-lang.xyz/rust/article/4-Cargo-사용법](http://rust-lang.xyz/rust/article/4-Cargo-%EC%82%AC%EC%9A%A9%EB%B2%95)

> 클라우드(공유 가상화) 환경에서는 실행 불가 → **로컬 가상화 (QEMU)** 환경에서 실습.

## 학습 목차

1. **Freestanding 실행파일** — `no_std` 환경에서 표준 라이브러리 없이 빌드, 패닉 핸들러 구현 → `3.Freestanding_실행파일.md`
2. **최소 기능을 갖춘 커널** — VGA 텍스트 버퍼 출력, BIOS 부팅, 부트로더 연결 → `4.최소_기능_커널.md`

## 보조 주제

- BIOS / UEFI 부팅 흐름 이해
- `no_std` 환경에서의 panic / runtime 처리 패턴
- Cargo의 `target.json` 커스텀 타겟 설정
