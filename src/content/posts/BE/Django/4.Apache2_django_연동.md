# 2. mod_wsgi 설치

```bash
sudo apt-get install libapache2-mod-wsgi-py3
```

# 3. VirtualHost 설정

```apache
<VirtualHost *:80>
    ServerAdmin webmaster@localhost
    ServerName myproject.com                       # 웹사이트 도메인
    DocumentRoot /path/to/your/django/project      # Django 프로젝트 루트

    WSGIDaemonProcess myproject python-home=/path/to/your/venv python-path=/path/to/your/django/project
    WSGIProcessGroup myproject
    WSGIScriptAlias / /path/to/your/django/project/myproject/wsgi.py process-group=myproject

    <Directory /path/to/your/django/project>
        Options Indexes FollowSymLinks MultiViews
        AllowOverride All
        Require all granted
    </Directory>

    ErrorLog ${APACHE_LOG_DIR}/error.log
    CustomLog ${APACHE_LOG_DIR}/access.log combined
</VirtualHost>
```

> 가상환경을 사용하지 않는 경우 `python-home`을 제거한다.
