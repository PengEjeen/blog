# 2. 핵심 흐름

1. `models.py`의 모델에 `FileField` 또는 `ImageField` 추가
2. `settings.py`에서 `MEDIA_URL`, `MEDIA_ROOT` 설정
3. form에서 `enctype="multipart/form-data"`로 업로드
4. view에서 `request.FILES`로 파일 객체 획득
5. 모델 인스턴스 저장 시 `upload_to` 경로에 자동 저장

```python
# models.py
class Document(models.Model):
    file = models.FileField(upload_to="uploads/%Y/%m/%d/")
    uploaded_at = models.DateTimeField(auto_now_add=True)
```

```python
# settings.py
MEDIA_URL = '/media/'
MEDIA_ROOT = os.path.join(BASE_DIR, 'media')
```
