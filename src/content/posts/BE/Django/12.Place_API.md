# 0. 자료

- [Google Places Nearby Search (new)](https://developers.google.com/maps/documentation/places/web-service/nearby-search?hl=ko)

# 1. 주변 검색

## 1.1 Request format

```python
json_data = {
    "includedTypes": ["cafe"],          # restaurant, chinese_restaurant, cafe 등
    "maxResultCount": 5,                 # 최대 20
    "rankPreference": "DISTANCE",        # DISTANCE 또는 POPULARITY
    "locationRestriction": {
        "circle": {
            "center": {
                "latitude": geometry["lat"],
                "longitude": geometry["lng"],
            },
            "radius": 500.0              # 0.0 이상 50000.0 이하 (m)
        }
    }
}
```

## 1.2 Return format (예시)

응답 JSON에는 `places[]` 배열이 들어 있고, 각 place는 다음 필드들을 포함한다.

| 필드 | 설명 |
|------|------|
| `name` | `places/{place_id}` 형식 리소스명 |
| `id` | place_id |
| `types` | 카테고리 배열 (예: `seafood_restaurant`, `restaurant`, ...) |
| `nationalPhoneNumber` / `internationalPhoneNumber` | 전화번호 |
| `formattedAddress` | 주소 (사람이 읽는 형식) |
| `addressComponents` | street_number / route / locality / country 등 구성요소 |
| `location` | `{ latitude, longitude }` |
| `viewport` | bounding box (`low`, `high`) |
| `rating` | 별점 (0~5) |
| `googleMapsUri` / `websiteUri` | 외부 링크 |
| `regularOpeningHours.periods[]` | 요일/시간 영업정보 (`day`: 0=일요일) |
| `photos[]` | 사진 리소스 + author attribution |
| `outdoorSeating`, `servesCoffee`, `servesCocktails`, ... | 부대 정보 (boolean) |
| `paymentOptions` | `acceptsDebitCards`, `acceptsNfc` 등 |
| `parkingOptions` | `paidParkingLot`, `paidStreetParking` 등 |
| `accessibilityOptions` | 휠체어 접근성 정보 |

```json
{
  "places": [
    {
      "name": "places/ChIJURDKN2eAhYARN0AMzUEaiKo",
      "id": "ChIJURDKN2eAhYARN0AMzUEaiKo",
      "types": ["seafood_restaurant", "restaurant", "food", "point_of_interest", "establishment"],
      "formattedAddress": "PIER 1 1/2 The Embarcadero N, San Francisco, CA 94105, USA",
      "location": { "latitude": 37.7971645, "longitude": -122.3953359 },
      "rating": 4.5,
      "googleMapsUri": "https://maps.google.com/?cid=12288100453195726903",
      "regularOpeningHours": { "openNow": false, "periods": [/* ... */] },
      "outdoorSeating": true,
      "servesCoffee": true,
      "paymentOptions": { "acceptsDebitCards": true, "acceptsNfc": true },
      "accessibilityOptions": { "wheelchairAccessibleEntrance": true }
    }
  ]
}
```

> 응답 필드는 `X-Goog-FieldMask` 헤더로 제한할 수 있다 (필요한 필드만 받기).

# 2. 자동완성 (Autocomplete)

> 2024-05-03 기준: 구글에서 테스트 중인 기능.

## 2.1 Request format

```python
json_data = {
    "input": "travel",
    "includedPrimaryTypes": ["cafe"],
    "locationRestriction": {
        "circle": {
            "center": {
                "latitude": geometry["lat"],
                "longitude": geometry["lng"],
            },
            "radius": 50000.0   # 0.0 이상 50000.0 이하
        }
    }
}
```

## 2.2 Return format

`X-Goog-FieldMask` 헤더로 받을 필드를 지정한다.

```
X-Goog-FieldMask: suggestions.placePrediction.placeId
```
