title: 10장 Vue + Apache 연동
date: 2026-04-24
---

빌드된 Vue 정적 파일을 Apache로 서빙하고, `/api/` 경로는 Django 백엔드로 프록시한다.

```apache
<VirtualHost *:80>
    ServerName your_domain.com
    DocumentRoot /path/to/your/vue/app/dist

    <Directory /path/to/your/vue/app/dist>
        Options Indexes FollowSymLinks
        AllowOverride All
        Require all granted
    </Directory>

    ProxyPass /api/ http://backend_server:8000/
    ProxyPassReverse /api/ http://backend_server:8000/
</VirtualHost>
```

# 핵심

- `DocumentRoot`: Vue `npm run build` 결과물 (`dist/`).
- `ProxyPass /api/`: Vue 앱에서 `/api/...` 호출 시 Django 백엔드로 라우팅.
- `mod_proxy`, `mod_proxy_http` 활성화 필요:

  ```bash
  sudo a2enmod proxy proxy_http
  sudo systemctl restart apache2
  ```
