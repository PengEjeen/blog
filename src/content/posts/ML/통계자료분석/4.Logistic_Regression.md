# 1. Logistic Regression

이항 반응변수(`0/1`)에 대해 사건 발생 확률을 모델링한다.

## 1.1 선형회귀의 한계
- 선형회귀로 확률을 직접 모델링하면 예측값이 `[0,1]` 범위를 벗어날 수 있다.

## 1.2 로지스틱 회귀
- 확률 `p`를 logit으로 변환해 선형 결합으로 모델링한다.

`logit(p) = log(p/(1-p)) = beta0 + beta1 x1 + ... + betak xk`

- 추정은 OLS가 아니라 최대우도추정(MLE)을 사용한다.
- 반복 알고리즘(Fisher scoring/IRLS)으로 계수를 추정한다.

### complete separation
- 설명변수 조합이 반응변수를 완벽 분리하면 추정 실패/불안정이 발생할 수 있다.
- 희귀사건, 작은 표본, 특정 범주 쏠림에서 자주 발생한다.

# 2. 적합도와 추론

## 2.1 Deviance
- `Deviance = -2 * log(likelihood)`
- 값이 작을수록 적합이 좋다.

## 2.2 Null vs Residual deviance
- Null deviance: 절편만 있는 모형
- Residual deviance: 현재 모형
- 차이를 이용해 전체 모형 유의성을 카이제곱 검정으로 확인한다.

## 2.3 계수 해석
- 계수의 지수값 `exp(beta)`는 odds ratio.
- `OR > 1`: 사건 odds 증가
- `OR < 1`: 사건 odds 감소
- 신뢰구간이 1을 포함하면 유의성이 약함

# 3. 변수 선택

로지스틱에서는 전통적인 `Adjusted R^2`, `Cp` 활용이 제한적이므로 아래를 주로 사용한다.
- AIC/BIC
- stepAIC (forward/backward/hybrid)
- best subset
- Lasso

Lasso:
- 람다 증가 시 계수 축소
- 일부 계수는 0으로 수축되어 변수 선택 효과
- 교차검증으로 최적 람다 선택

# 4. 진단

- Pearson residual / Deviance residual
- 선형성 점검(연속형 설명변수)
- 다중공선성(VIF)
- 이상치/영향점

---

# 5. 분류 성능 평가

분류 기준값 `d`로 확률을 클래스화한다.

주요 지표:
- Accuracy
- Sensitivity(Recall, TPR)
- Specificity(TNR)
- Precision
- F1
- ROC-AUC

참고:
- Accuracy 단독 평가는 불균형 데이터에서 오판 가능성이 높다.
- 목적(예: FN 최소화/FP 최소화)에 따라 기준값 조정이 필요하다.

---

# 6. 희귀사건 분류

희귀사건에서는 기본 `d=0.5`가 민감도 저하를 유발하기 쉽다.

대응:
1. 임계값 튜닝
2. 샘플링 전략
- Up-sampling
- Down-sampling
- SMOTE(주로 연속형 특성에서 사용)

실무에서는 ROC/PR 기반으로 threshold를 정하고, 비용 함수(오탐/미탐 비용)를 반영해 결정한다.
