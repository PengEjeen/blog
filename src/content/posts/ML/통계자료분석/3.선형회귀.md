# 회귀모형의 적합

## 1. 단순회귀모형
- 오차항이 평균 0, 분산 일정, 독립이라는 가정에서 최소제곱추정량(OLS)을 사용한다.
- RSS를 최소화하는 회귀계수 `(beta0, beta1)`를 찾는다.
- 가우스-마코프 정리 하에서 OLS는 BLUE이다.

## 2. 다중회귀모형
- 설명변수가 여러 개인 경우 RSS 최소화로 계수를 추정한다.
- 모형 적합 전 상관계수/산점도행렬로 변수 관계를 점검한다.
- `lm` 객체에서 주요 확인 항목:
  - `anova`, `coefficients`, `confint`, `fitted`, `residuals`, `summary`

## 3. 다항회귀모형
- 선형 관계가 아닌 경우 다항식 항을 추가한다.
- 차수는 과대복잡도를 피하도록 가능한 낮게 선택한다.
- 범주형 변수는 더미 변수(`범주-1`)로 반영한다.

# 회귀모형의 추론

## 1. 모형 유의성 검정 (F-test)
가설:
- `H0: beta1 = ... = betak = 0`
- `H1: 적어도 하나는 0이 아님`

검정통계량:
- `F = ((TSS - RSS)/k) / (RSS/(n-k-1))`

의미:
- `TSS`: 총변동
- `RSS`: 회귀모형으로 설명되지 않은 변동
- `TSS - RSS`: 회귀모형이 설명한 변동

## 2. 개별 계수 검정
- 각 계수에 대해 t-검정/신뢰구간으로 유의성 판단.

## 3. 확장모형 vs 축소모형 비교
- 필요 없는 변수를 제거한 단순 모형과 비교해 설명력 대비 복잡도를 평가.

# 변수 선택

## 1. 평가 지표
- `R^2`
- `Adjusted R^2`
- `RSE`
- `AIC`
- `BIC`
- `CV error`

## 2. 탐색 전략
- Forward selection
- Backward elimination
- Hybrid stepwise
- Best subset selection

## 3. Shrinkage
- Ridge: 계수를 0에 가깝게 축소(완전 제거는 아님)
- Lasso: 일부 계수를 0으로 만들어 변수 선택 효과

# 회귀진단

점검 가정:
1. 오차 평균 0
2. 등분산성
3. 정규성
4. 독립성
5. 선형성

주요 플롯:
1. Residuals vs Fitted
2. Normal Q-Q
3. Scale-Location
4. Residuals vs Leverage

추가 점검:
- 다중공선성: `VIF` (보통 10 이상이면 위험 신호)
- 영향점/이상점: leverage, 표준화 잔차, Cook's distance
- 부분잔차 플롯(`crPlots`)로 선형성 재확인
