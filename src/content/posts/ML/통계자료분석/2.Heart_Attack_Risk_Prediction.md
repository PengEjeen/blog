# Heart Attack Risk Prediction
201952036 조남영

심장마비 위험(이진 반응변수)을 연령, 콜레스테롤, 혈압, 생활습관 등 임상/행동 변수로 예측하는 로지스틱 회귀 기반 분석.

데이터 출처:
- https://www.kaggle.com/datasets/iamsouravbanerjee/heart-attack-prediction-dataset/code

---

## 1. 데이터 확인

- 관측치: 8,763
- 변수: 26개(식별자 포함)
- 반응변수: `Heart.Attack.Risk` (0/1)

주요 변수:
- Age, Sex, Cholesterol, Blood Pressure, Heart Rate
- Diabetes, Family History, Smoking, Obesity
- Alcohol Consumption, Exercise Hours, Diet
- Previous Heart Problems, Medication Use
- Stress Level, Sedentary Hours, Income
- BMI, Triglycerides, Physical Activity Days
- Sleep Hours, Country, Continent, Hemisphere

전처리:
1. `Patient.ID` 제거
2. 범주형 변수 factor 변환
3. `Blood.Pressure`를 수축기/이완기로 분리
4. `Country` 제거, `Continent` 유지
5. 결측값 확인: 없음

반응변수 비율:
- 사건(1): 약 0.358
- 비사건(0): 약 0.642

희귀사건에 가까운 불균형 구조를 보인다.

---

## 2. 모형 선택

데이터 분할:
- train/test = 0.8 / 0.2

초기 전체 모형(`glm(binomial)`)에서 유의성이 높지 않은 변수가 다수 확인됨.

변수선택 기법 적용:
1. `bestglm` (AIC/BIC)
2. `stepAIC` (forward/backward/hybrid)
3. `Lasso` (`cv.glmnet`)

후보 모형:
- 후보 A: `Heart.Rate + Cholesterol + Diabetes + Sleep.Hours.Per.Day`
- 후보 B: `Heart.Rate + Cholesterol + Sleep.Hours.Per.Day`

비교 결과:
- AIC는 후보 A가 약간 유리
- BIC는 후보 B가 더 유리
- 최종 선택: 후보 B (간결성/안정성 고려)

최종 모형:
`Heart.Attack.Risk ~ Cholesterol + Heart.Rate + Sleep.Hours.Per.Day`

## 3. 회귀 진단

점검 항목:
- 잔차 플롯
- 선형성 검정
- 다중공선성(VIF)

결과 요약:
- VIF는 위험 수준 아님(중대한 다중공선성 미확인)
- 선형성은 큰 위배 증거 없음
- 잔차 패턴은 실무적으로 수용 가능 범위

---

## 4. 예측 및 분류 성능

### 4.1 기본 기준값(0.5)
- 정확도는 높아 보이나,
- 민감도가 0에 가까워 사건 탐지 실패가 발생.

### 4.2 임계값 튜닝
- validation set에서 sensitivity/specificity 균형 기준으로 threshold 재설정(약 0.34)
- 재예측 시 민감도는 개선되나 특이도/정확도와 trade-off 발생

### 4.3 ROC/AUC
- ROC 기반으로 임계값 선택 타당성 확인
- 심장질환 예측 맥락에서는 FN(놓침) 비용이 커서 민감도 우선 전략이 합리적

결론:
- 보수적 임계값 조정으로 사건 탐지 성능을 개선하는 접근이 유효
- 단일 정확도보다 민감도/특이도 균형 지표로 평가해야 함

## Appendix A (Heart Attack 분석 코드 핵심)
```r
library(dplyr)
library(bestglm)
library(MASS)
library(glmnet)
library(car)
library(caret)
library(pROC)

df <- read.csv("heart_attack_prediction_dataset.csv")

# 전처리
df <- df %>%
  mutate(
    Diabetes = as.factor(Diabetes),
    Family.History = as.factor(Family.History),
    Sex = as.factor(Sex),
    Smoking = as.factor(Smoking),
    Obesity = as.factor(Obesity),
    Alcohol.Consumption = as.factor(Alcohol.Consumption),
    Diet = as.factor(Diet),
    Previous.Heart.Problems = as.factor(Previous.Heart.Problems),
    Medication.Use = as.factor(Medication.Use),
    Heart.Attack.Risk = as.factor(Heart.Attack.Risk)
  ) %>%
  select(-Patient.ID, -Country, -Hemisphere, -Continent)

bp <- data.frame(do.call('rbind', strsplit(df$Blood.Pressure, "/")))
df$Blood.Pressure.systolic <- as.integer(bp$X1)
df$Blood.Pressure.diastolic <- as.integer(bp$X2)
df <- df %>% select(-Blood.Pressure)

set.seed(201952036)
idx <- createDataPartition(df$Heart.Attack.Risk, p=0.8, list=FALSE)[,1]
train_df <- df %>% slice(idx)
test_df  <- df %>% slice(-idx)

fit_full <- glm(Heart.Attack.Risk ~ ., train_df, family=binomial)
fit_null <- glm(Heart.Attack.Risk ~ 1, train_df, family=binomial)

fit_step <- stepAIC(fit_full, direction="both", trace=FALSE)

fit_final <- glm(
  Heart.Attack.Risk ~ Cholesterol + Heart.Rate + Sleep.Hours.Per.Day,
  train_df,
  family=binomial
)

pred_test <- predict(fit_final, newdata=test_df, type="response")
yhat_05 <- factor(if_else(pred_test >= 0.5, "1", "0"))

cm_05 <- confusionMatrix(reference=test_df$Heart.Attack.Risk, data=yhat_05, positive="1")

roc_obj <- roc(test_df$Heart.Attack.Risk, pred_test)
auc_val <- auc(roc_obj)
```
