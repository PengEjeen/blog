# SVM

## 1. 개요
- SVM(Support Vector Machine)은 분류, 회귀, 이상치 탐지에 사용할 수 있는 다목적 모델이다.
- 작은/중간 규모 데이터셋에서 강력한 성능을 보인다.
- 선형/비선형 경계를 모두 다룰 수 있다.

## 2. 선형 SVM

### 2.1 기본 아이디어
- 목표는 클래스를 가장 넓은 마진으로 분리하는 초평면을 찾는 것이다.
- 마진 경계에 걸리는 샘플들을 `Support Vector`라고 한다.
- 특성 수가 `n`이면 결정함수는 `n`차원 초평면이고 경계는 `n-1`차원이다.

결정함수:
`f(x) = w^T x + b`

예측:
- `f(x) < 0`이면 클래스 0
- `f(x) >= 0`이면 클래스 1

### 2.2 목적함수

#### 2.2.1 하드 마진
- 데이터가 완전히 선형 분리 가능하고 마진 위반이 없는 경우.
- 목적: `1/2 * ||w||^2` 최소화.
- 제약: 모든 샘플이 마진 바깥에 위치.

#### 2.2.2 소프트 마진
- 실제 데이터의 노이즈/이상치를 허용하기 위해 슬랙 변수 `zeta_i` 도입.
- 목적: `1/2 * ||w||^2 + C * sum(zeta_i)` 최소화.
- `C`는 마진 확대와 오분류 패널티 사이의 균형을 조절한다.

## 3. 비선형 SVM

### 3.1 다항식 커널
- 고차원 다항 특성을 직접 만들지 않고 커널 트릭으로 동일 효과를 얻는다.
- `SVC(kernel='poly')` 등으로 적용 가능.

### 3.2 유사도 특성
- 랜드마크와의 유사도를 새 특성으로 만들 수 있다.
- 대표 함수: Gaussian RBF

`similarity(x, l) = exp(-gamma * ||x-l||^2)`

### 3.3 가우시안 RBF 커널
- 유사도 특성을 대량 생성하지 않고 커널 트릭으로 계산 비용을 줄인다.
- `gamma`가 크면 경계가 복잡해지고, 작으면 부드러워진다.

# Decision Tree

## 1. 개요
- 분류/회귀/다중출력 문제에 사용 가능.
- 전처리 부담이 작고 해석성이 좋다.
- 핵심은 분할 기준을 반복 적용해 불순도를 줄이는 것.

## 2. 분기 기준

### 2.1 CART
- 이진 분할만 수행.
- 각 노드에서 불순도 감소가 가장 큰 분할을 greedy하게 선택.

### 2.2 ID3
- 엔트로피 기반 정보이득 최대화.
- `criterion='entropy'`로 사용.

## 3. 규제 하이퍼파라미터
- `max_depth`: 최대 깊이
- `min_samples_split`: 분할 최소 샘플
- `min_samples_leaf`: 리프 최소 샘플
- `max_leaf_nodes`: 리프 최대 개수
- `max_features`: 분할 시 고려 특성 수

# RandomForestClassifier

## 1. 개요
- 여러 결정트리를 배깅/무작위 특성 선택으로 앙상블한 모델.
- 분산을 줄여 일반화 성능을 개선한다.

## 2. 학습 절차
1. 부트스트랩 샘플링으로 학습 데이터 생성
2. 각 샘플로 결정트리 학습
3. 각 노드 분할 시 무작위 특성 부분집합에서 최적 분할 선택
4. 모든 트리 결과를 다수결(분류) 또는 평균(회귀)으로 집계

## 3. 주요 하이퍼파라미터
- `n_estimators`
- `max_features`
- `max_depth`
- `min_samples_split`
- `min_samples_leaf`
- `bootstrap`
- `criterion`
- `oob_score`
- `class_weight`

# Boosting

## 1. AdaBoost
- 이전 모델이 틀린 샘플 가중치를 높여 다음 약분류기가 보정하도록 학습.
- 최종 예측은 각 분류기의 가중합으로 결정.
- 하이퍼파라미터: `n_estimators`, `learning_rate`, `algorithm`.

## 2. Gradient Boosting
- 샘플 가중치 대신 이전 모델의 잔차를 다음 트리가 학습.
- 순차적으로 오차를 줄여 성능 향상.

## 3. XGBoost
- Gradient Boosting + 강한 정규화(L1/L2), 빠른 학습 최적화.
- 주요 파라미터: `learning_rate`, `n_estimators`, `max_depth`, `min_child_weight`, `subsample`, `colsample_bytree`.

## 4. LightGBM
- 리프 중심(leaf-wise) 성장으로 대규모 데이터에서 빠르고 정확.
- 주요 파라미터: `learning_rate`, `n_estimators`, `max_depth`, `min_data_in_leaf`, `bagging_fraction`, `feature_fraction`.

## 5. CatBoost
- 범주형 변수 처리에 강점.
- 순서 보존 방식으로 과적합 완화.
- 주요 파라미터: `learning_rate`, `n_estimators`, `depth`, `cat_features`, `l2_leaf_reg`.
