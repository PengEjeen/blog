# 시계열 분석 (인덱스)

시계열 분석 학습 정리. 이론 + 실제 환율 데이터 케이스로 마무리.

## 1. 시계열 모델 (ETS, ARIMA)

상세 → `ML/시계열/2.시계열_모델_ETS_ARIMA.md`

핵심 키워드:
- **정상성** — 평균/분산이 시간에 무관해야 ARMA 류 모델 적용 가능
- **차분** — 차분으로 추세/계절성 제거 후 정상화
- **ACF / PACF** — AR/MA 차수 결정에 활용
- **ETS / ARIMA / Regression with ARMA errors** — 모델군 비교
- **모델 진단** — 잔차의 백색잡음성 검정 (Ljung-Box 등)
