# ML (머신러닝 기초 실습 인덱스)

데이터 → 전처리 → 학습 → 평가 → 기록의 흐름을 다양한 데이터셋으로 반복 연습한 메모.

## 1. Housing 실습 (예제)

- 캘리포니아 주택 가격 예측 (회귀)
- 파이프라인: imputer → scaler → regressor
- 산출물: `Housing.ipynb`, `Housing.py`

## 2. Wine-red 실습 (개인)

- UCI Wine Quality (red) 분류
- 데이터 불균형 처리 / 트리 계열 모델 비교
- 산출물: `winequality-red.csv`, `Wine-red.ipynb`

## 3. MNIST 실습 (예제)

- 손글씨 숫자 분류 (다중 클래스)
- KNN, SGD, 랜덤포레스트 비교
- 산출물: `MNIST.ipynb`

## 4. 모델훈련 (예제)

- 학습/검증 분리, 교차검증, 하이퍼파라미터 튜닝 (GridSearchCV / RandomizedSearchCV)
- 산출물: `모델훈련.ipynb`

## 5. 서포트 벡터 머신 (SVM)

- 선형/RBF 커널 비교, 마진 / 슬랙 변수 이해
- 산출물: `SVM.ipynb`

## 6. LLM (확장)

- ML 흐름의 마지막으로 다룬 LLM 기초 → `ML/기초/2.LLM.md`
- 참고 - [https://seungbeomdo.tistory.com/79](https://seungbeomdo.tistory.com/79)

## 공통 흐름

데이터 탐색 → 전처리 → 모델 학습 → 검증/평가 → 실험 기록.
