# 2.분할정복 알고리즘

개요

문제의 크기가 매우 큰 경우

- 분할된 입력에 대한 문제를 부분 문제
- 부분 문제의 해를 부분해
- 부분 문제는 더 이상 분할할 수 없을 때까지 분할

하향식 문제풀이

문제해결 단계

1. 주어진 입력을 둘 이상의 부분 문제로 분할 후 각 부분 문제를 더욱 작은 크기로 분할.
2. 분할할 수 없을 때까지 분할 후 부분해
3. 구해진 부분해들의 조합을 통해 원래 주어진 문제의 해를 구함

분할횟수

k = log2(n)

분할정복 알고리즘의 기본

부분문제들의해로상위 부분문제 해결

분할 정복 알고리즘의 분류

- 분할되는 부분 문제의 수 a 와 부분 문제의 크기 b(감소량 1/b) 에 따라 분류
- a=b=2 **합병정렬**(최근접 정의 쌍 찾기, 공제선 문제)
- a=3, b=2 큰 정수의 곱셈
- a=4, b=2 큰 정수의 곱셈
- a=7, b=2 스트라센의 행렬 곱셈 알고리즘
- a=2 1개의 문제에 대해서는 고려할 필요가 없으며, 부분문제의 크기가 1/2로 감소하는 알고리즘: **이진탐색**
- **a=2. 1개의 부분문제는 고려할 필요 없으며. 부분문제의 크기가 일정하지 않은 크기로 감소하는 알고리즘: 선택문제**
- **부분문제의 크기가 1.2개씩 감소하는 알고리즘: 삽입정렬, 피보니치 수**

## 2.1 합병정렬(merge sort)

입력이 2개의 부분문제로 분할, 부분 문제의 크기가 1/2로 감소

방식

1. n개의 숫자들을 n/2개씩 2개의 부분 문제로 분할
2. 각각의 부분 문제를 재귀적으로 합병 정렬
3. 2개의 정렬된 부분을 합병하여 정렬
4. 합병 과정이 문제를 정복

시간복잡도 O(nlogn)

- 정렬의 시간 복잡도는 일반적으로 숫자의 비교 횟수로 표현
- 합병의 수행 시간은 입력의 크기에 비례

응용

- 외부 정렬(external sort)의 기본이 되는 정렬 알고리즘
- 퀵이나 힙 정렬보다 효율적
- 멀티코어 CPU와 GPU 등장으로 **정렬 알고리즘을 병렬화하는 데에 합병 정렬 알고리즘 활용**

단점

- 합병 정렬의 공간 복잡도 O(n)
- 입력을 위한 메모리 공간 외에 **추가로 입력과 같은 크기의 공간이 별도 필요**

## 2.2퀵 정렬(quick sort)

데이터를 분할하고 분할된 부분 별로 이동하는 정복 과정을 반복하여 거쳐 정렬하는 방법

방식

1. 피봇이라 일컫는 배열의 원소 기준 작은 것은 왼편 큰 것은 오른편으로 이동
2. 피봇을 그 사이에 배치
3. 분할된 문제들에 대해서도 동일한 과정 순환호출
4. 부분배열 분할 불가능할 때까지 반복

시간복잡도 O(nlog2(n))

퀵 정렬 성능 향상 방법

- 삽입 정렬이 동시에 사용
- 입력 크기가 작을 때에는 삽입정렬보다 빠르지만은 않다.
- 따라서 부분문제의 크기가 작아지면 분할 중단하고 삽입정렬 사용하게 됨.

퀵 정렬의 응용

- 커다란 크기의 입력에 대해서 가장 좋은 성능

## 2.3선택문제

선택문제

- 최대 / 최소 숫자를 찾는 문제
- k-번째 숫자를 찾는 문제

방식

이진 탐색의 개념 적용 == 1/2로 분할한 두 부분 중에서 한 부분만을 검색

- 퀵 정렬의 피봇 개념 적용
- 숫자들 중 피봇을 선택하여 그림과 같이 분할
- 분할 후, 각 그룹의 크기를 알아야… k 번째 작은 숫자가 어느 그룹에 있는지 알 수 있고, 그 다음엔 그 그룹에서 몇 번째로 작은 숫자를 찾아야 하는지 알 수 있음.
1. 분할 후 찾고자 하는 k번째 작은(큰) 숫자의 탐색
2. 문제가 2개의 부분문제로 분할되나 1개의 부분문제는 고려할 필요 없음. 문제의 크기가 일정하지 않은 크기로 감소

시간복잡도 O(n)

피봇 랜덤하게 정했을 때 good 분할이 될 확률 ½

## 2.4최근접 점쌍 알고리즘(closest pair)

거리가 가장 가까운 한 쌍의 점을 찾는 문제

시간복잡도O(n^2)

방법

- 왼쪽 부분의 최근접 점의 쌍 사이의 거리, 오른쪽 부분의 최근접 점의 쌍 사이의 거리
- 배열에는 점들이 x좌표의 오름차순으로 정렬 y좌표는 생략

입력 - x좌표의 오름차순으로 정렬된 배열 S

1. 정렬된 S를 같은 크기의 SL, SR로 분할, 홀수면 SL = SR+1
2. ClosetPair(SL)
3. ClosetPair(SR)
4. 중간영역에 속하는 점들 중에서 최근점 점의 쌍 CPc.

분할정복 주의할 점

부적절한 경우

- 분할된 부분 문제의 입력 크기의 합이 분할되기 전의 입력 크기보다 매우 커지는 경우
