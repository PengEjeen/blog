# Assembly (x86_64)

참고 교재 - [UNLV assembly64.pdf](http://www.egr.unlv.edu/~ed/assembly64.pdf)

> Rust OS 실습에서 부팅 단계 / freestanding 실행파일 이해를 위해 함께 정리한 학습 메모.

## 학습 목차

1. **Hello World** — `_start` / `sys_write` / `sys_exit` 시스템콜 호출
2. **기본 문법** — 레지스터(rax, rbx, rcx, rdx, rsi, rdi…), 메모리 주소 지정 모드
3. **키보드 입력 받아 출력해보기** — `sys_read` 로 stdin 읽고 stdout 으로 echo
4. **for 반복** — `cmp` + `jmp/jne/jl` 로 루프 구성
5. **macro** — NASM 매크로 정의와 인라인 확장
6. **sys_open / sys_close** — 파일 디스크립터 기반 I/O
7. **디어셈블링** — `objdump -d` 로 바이너리 역분석

## 보조 주제

- 함수 호출 규약 (System V AMD64 ABI: rdi, rsi, rdx, rcx, r8, r9 → 인자 / rax → 반환)
- 스택 프레임과 `rbp/rsp` 사용
- 저수준 디버깅 관점 (gdb, layout asm)
