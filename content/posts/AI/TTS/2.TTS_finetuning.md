# TTS finetuning

사용 모델: `tts_models--multilingual--multi-dataset--xtts_v2`

## 1. Preprocessing

audio directory = `wavs` (임시)

### 1.1 datasets

dataset 디렉토리 구조:

```
metadata.txt
/wavs
 |- audio1.wav
 |- audio2.wav
```

`metadata.txt` 형식:

```
{audiofile_name1}|{text}
{audiofile_name2}|{text}
```

## 2. Training

```bash
git clone https://github.com/coqui-ai/TTS
cd TTS
pip install -e .
```

1. `TTS/bin/train_tts.py`로 학습 진행
2. 학습용 `config.json` 작성

### config.json 예시

```json
{
  "run_name": "my_run",
  "model": "glow_tts",
  "batch_size": 32,
  "eval_batch_size": 16,
  "num_loader_workers": 4,
  "num_eval_loader_workers": 4,
  "run_eval": true,
  "test_delay_epochs": -1,
  "epochs": 1000,
  "text_cleaner": "english_cleaners",
  "use_phonemes": false,
  "phoneme_language": "en-us",
  "phoneme_cache_path": "phoneme_cache",
  "print_step": 25,
  "print_eval": true,
  "mixed_precision": false,
  "output_path": "recipes/ljspeech/glow_tts/",
  "datasets": [
    {
      "formatter": "ljspeech",
      "meta_file_train": "metadata.csv",
      "path": "recipes/ljspeech/LJSpeech-1.1/"
    }
  ]
}
```

### 학습 실행

```bash
CUDA_VISIBLE_DEVICES="0" python TTS/bin/train_tts.py \
  --config_path /home/ubuntu/.local/share/tts/tts_models--en--ljspeech--glow-tts/config.json \
  --restore_path /home/ubuntu/.local/share/tts/tts_models--en--ljspeech--glow-tts/model_file.pth
```

### 또는 recipe 스크립트로 실행

`recipes/ljspeech/glow_tts/train_glowtts.py`로 학습:

```bash
CUDA_VISIBLE_DEVICES="0" python recipes/ljspeech/glow_tts/train_glowtts.py \
  --restore_path /home/ubuntu/.local/share/tts/tts_models--en--ljspeech--glow-tts/model_file.pth \
  --coqpit.run_name "glow-tts-finetune" \
  --coqpit.lr 0.00001
```
