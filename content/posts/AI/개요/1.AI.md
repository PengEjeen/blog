# AI

AI/LLM 관련 학습·실험·구현 정리. 이 카테고리는 다음과 같은 하위 폴더로 구성:

## 폴더 구성

- **개요/** — 본 인덱스 + 응용(FastAPI 서빙) + Optimizer / Gemma 개요
- **LangGraph/** — LangGraph 그래프 워크플로우 + map-reduce 병렬 패턴
- **LLM/** — LLM 환경/파인튜닝, RNN 기초부터 Llama3·RAG까지
- **TTS/** — XTTS2 기반 음성 합성 파인튜닝
- **논문정리/** — 직접 정리한 논문 리뷰들

## 주요 키워드

- LLM 학습 환경 구성 (CUDA / cuDNN / WSL / bitsandbytes)
- 파인튜닝 (Full / LoRA / QLoRA / SFT / RLHF)
- LangGraph 상태 기반 그래프, map-reduce 병렬 실행
- RAG (Retrieval-Augmented Generation) 구현
- TensorRT-LLM 기반 Llama3 추론 최적화
- TTS finetuning (XTTS2)
