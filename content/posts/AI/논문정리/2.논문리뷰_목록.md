# 논문리뷰

읽고 정리한 LLM/생성 모델 관련 주요 논문 모음.

## 1. Latent Diffusion for Language Generation

- 리뷰 - [kimjy99 블로그](https://kimjy99.github.io/%EB%85%BC%EB%AC%B8%EB%A6%AC%EB%B7%B0/ldlg/)
- 핵심 아이디어: 텍스트 생성에 latent diffusion 적용. 픽셀 공간이 아닌 잠재 공간에서 디퓨전 → 효율 향상.

## 2. Gemma 2: Improving Open Language Models at a Practical Size

- Google DeepMind, 2024
- 핵심: knowledge distillation + sliding window attention + soft logit capping 으로 작은 모델에서 큰 모델 수준 성능

## 3. Retrieval-Augmented Generation for Knowledge-Intensive NLP Tasks

- Lewis et al., 2020 (NeurIPS)
- 핵심: 파라메트릭 메모리(LLM) + 비파라메트릭 메모리(외부 검색) 결합 → 사실 정확도와 업데이트 가능성 향상
- 현재 모든 RAG 시스템의 기반이 되는 논문
