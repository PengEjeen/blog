# Retrieval-Augmented Generation for Knowledge-Intensive NLP Tasks

Lewis, P., Perez, E., Piktus, A., Petroni, F., Karpukhin, V., Goyal, N., ... & Kiela, D. (2020).
*Advances in Neural Information Processing Systems*, *33*, 9459-9474.
[paper](https://arxiv.org/abs/2005.11401)

## In Abstract

사전훈련된 매개변수 메모리(parametric)와 비매개변수 메모리(non-parametric)를 결합하는 모델인 **Retrieval-Augmented Generation (RAG)**의 범용 fine-tuning 방법 연구.

- **Parametric 메모리** — seq2seq 모델
- **Non-parametric 메모리** — dense vector index (외부 검색 가능한 지식 베이스)

## 목표

본 논문은 두 가지 RAG 변형을 비교한다:

- **RAG-Sequence Model** — 답변 시퀀스 전체에 대해 동일한 검색 결과 사용
- **RAG-Token Model** — 토큰 단위로 다른 문서를 검색해 사용

## 의의

- 모든 현대 RAG 시스템의 **기반이 되는 논문**
- 파라메트릭 메모리만으로는 어려운 **사실 정확도**, **지식 업데이트** 문제를 외부 검색 결합으로 풀어냄
- 이후 LangChain·LlamaIndex 같은 RAG 프레임워크들의 이론적 토대
