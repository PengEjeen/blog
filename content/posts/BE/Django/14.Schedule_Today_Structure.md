### Schedule

`schedule_id`는 `Random.new()`로 생성.

| 필드 | 타입 |
|------|------|
| `schedule_id` | `CharField(max_length=5)` — `SHA512(title)[:5]` |
| `title` | `CharField(max_length=20)` |
| `description` | `TextField()` — `default="None"` |
| `dateTimeofUpload` | `DateTimeField(auto_now=True)` |
| `cell_ids` | `JSONField(blank=True, null=True)` — `{"ids": []}` |

```json
cell_ids = { "ids": [] }
```

### Cell

`cell_id`는 `Random.new()`로 생성.

| 필드 | 타입 |
|------|------|
| `cell_id` | `CharField(max_length=7)` |
| `cell_mode` | `CharField(max_length=20)` — 페이지 name (`select_button`, `recommend_place`, `place_detail`) |
| `place_id` | `JSONField(blank=True, null=True)` — `{"place_id": []}` |

> `schedule_id`는 `Crypto.Random`으로 생성.
