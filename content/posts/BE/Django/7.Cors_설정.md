Django에서 CORS를 처리하려면 `django-cors-headers` 미들웨어를 사용한다.

# 1. 설치

```bash
pip install django-cors-headers
```

# 2. INSTALLED_APPS 추가

```python
# settings.py
INSTALLED_APPS = [
    # ...
    'corsheaders',
    # ...
]
```

# 3. MIDDLEWARE 추가

`SecurityMiddleware` 바로 아래에 추가하는 것이 권장된다.

```python
MIDDLEWARE = [
    # ...
    'django.middleware.security.SecurityMiddleware',
    'corsheaders.middleware.CorsMiddleware',
    # ...
]
```

# 4. CORS 설정

**모든 도메인 허용**

```python
CORS_ALLOW_ALL_ORIGINS = True
```

**특정 도메인만 허용**

```python
CORS_ALLOWED_ORIGINS = [
    "https://your-allowed-domain.com",
    "http://another-allowed-domain.com",
]
```

자세한 설정은 `django-cors-headers` 공식 문서 참조.

# 5. X-Frame-Options (iframe 허용)

`X-Frame-Options`를 `"DENY"`로 두면 iframe 렌더링이 차단된다. 특정 도메인에서 iframe 렌더링을 허용하려면 `XFrameOptionsMiddleware`를 비활성화한다.

```python
MIDDLEWARE = [
    # ...
    # 'django.middleware.clickjacking.XFrameOptionsMiddleware',
    # ...
]
```

> 보안상 가능하면 비활성화 대신 `X_FRAME_OPTIONS = 'SAMEORIGIN'` 또는 CSP `frame-ancestors`로 제어하는 것을 권장.
