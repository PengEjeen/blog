# 1. Request

```python
import googlemaps
import requests
from datetime import datetime

# Google Maps API 키
API_KEY = "YOUR_API_KEY"
gmaps = googlemaps.Client(key=API_KEY)

# 출발지/목적지 (place_id)
origin_place_id = 'ChIJ1yw4Rd6dfDURKxp9PlbLOdQ'
destination_place_id = 'ChIJKwRWVOadfDURJoaV-gLxkzc'

# Directions API URL
url = (
    f'https://maps.googleapis.com/maps/api/directions/json'
    f'?origin=place_id:{origin_place_id}'
    f'&destination=place_id:{destination_place_id}'
    f'&key={API_KEY}'
)

response = requests.get(url)
if response.status_code == 200:
    data = response.json()
```

> 주의: `origin=place_id%{...}` 처럼 URL 이스케이프가 잘못 들어가지 않도록 `place_id:` 접두사를 사용한다.

# 2. Response 구조 (요약)

응답 JSON의 핵심 키:

```text
{
  "geocoded_waypoints": [...],
  "routes": [
    {
      "bounds": { "northeast": {...}, "southwest": {...} },
      "copyrights": "...",
      "legs": [
        {
          "distance": { "text": "103 km", "value": 102980 },
          "duration": { "text": "1 hour 14 mins", "value": 4422 },
          "start_address": "Perth, UK",
          "end_address": "Glasgow, UK",
          "start_location": { "lat": 56.395, "lng": -3.430 },
          "end_location":   { "lat": 55.861, "lng": -4.258 },
          "steps": [
            {
              "distance": {...},
              "duration": {...},
              "html_instructions": "Head <b>east</b> on <b>South St</b>",
              "polyline": { "points": "wtewIja}S..." },
              "travel_mode": "DRIVING"
            }
            // ...
          ]
        }
      ],
      "overview_polyline": { "points": "..." },
      "summary": "A9 and M80",
      "warnings": [],
      "waypoint_order": []
    }
  ],
  "status": "OK"
}
```

# 3. 활용 포인트

- `legs[0].distance.value` (m) / `legs[0].duration.value` (sec) → 총 거리/소요시간 합산.
- `steps[].html_instructions` → 사용자에게 보여줄 턴-바이-턴 안내.
- `overview_polyline.points` → 전체 경로를 지도에 polyline으로 그릴 때 사용 (Google encoded polyline 포맷).
- `travel_mode` → DRIVING/WALKING/BICYCLING/TRANSIT.
