# 1. 시계열 분해

시계열은 보통 다음 요소로 분해한다.
- 추세(Trend): 장기 증가/감소 흐름
- 순환(Cycle): 비고정 주기의 반복 변동
- 계절(Seasonality): 고정 주기 반복
- 불규칙(Irregular): 설명되지 않는 잡음

가법형: `Y = T + S + C + I`
승법형: `Y = T * S * C * I`

## 정상 시계열 조건
1. 평균이 시간에 따라 일정
2. 분산이 시간에 따라 일정
3. 자기상관 구조가 시간 이동에 대해 불변

## 분해 방법
- 전통적 분해: 이동평균 기반 (`decompose`)
- STL 분해: Loess 기반 (`stl`)

# 2. ETS 모형

## 개요
- 지수평활 기반 예측 모형
- 최근 관측값에 더 큰 가중치를 두고 과거로 갈수록 지수적으로 감소
- 오차/추세/계절 성분 조합으로 모델을 체계적으로 선택

구성 축:
- Error: A, M
- Trend: N, A, Ad
- Seasonality: N, A, M

## 오차 측도
- MAE
- RMSE
- MAPE (0 근처 값에 취약)
- MASE (naive 대비 상대 성능 해석 가능)

## 모형 선택/진단
- 추정: MLE
- 비교 지표: AIC, AICc, BIC
- 변환: Box-Cox / 역변환
- 잔차 점검:
  - 독립성: ACF, Portmanteau/Ljung-Box
  - 평균 0 여부: 시계열/히스토그램
  - 등분산성: 시계열 플롯
  - 정규성: 히스토그램/Q-Q

# 3. ARIMA 모형

## 개요
- 정상 시계열을 대상으로 하는 확률모형
- 비정상 자료는 차분/변환으로 정상화 후 적용

## 정상성 확인
- 시계열 플롯
- ACF/PACF
- 단위근 검정
  - KPSS: 귀무가설이 정상성
  - ADF/PP: 귀무가설이 비정상성
- 차분 횟수 함수
  - `ndiffs()`
  - `nsdiffs()`

## 정상화
1. 분산 안정화
- 로그, 제곱근, Box-Cox

2. 수준 안정화(추세/계절 제거)
- 일반 차분
- 계절 차분

## 모형 식별
- ACF/PACF로 p, q 후보 탐색
- `auto.arima()`로 AICc 최소 모형 탐색
- 절편 포함 여부는 d, D와 함께 판단

# 4. 시계열 회귀 (ARMA Error)

## 4.1 Regression with ARMA error
- 반응변수는 시계열, 설명변수는 시점/계절 변수
- 계절성 표현:
  - seasonal dummy
  - Fourier series
- 오차는 ARMA로 모델링
- 핵심은 잔차에 구조가 남지 않도록 만드는 것

## 4.2 Dynamic regression with ARMA error
- 반응/설명 모두 시계열
- 모든 변수의 정상성 확보가 중요
- 한 변수라도 차분 필요 시 전체 일관 차분을 고려

---

# 5. 계절형 ARIMA 메모
1. 정상화 단계
- 분산 안정화
- 일반/계절 차분 차수 결정
2. 차수 결정
- AR(p,P), MA(q,Q)
3. 절편 포함 여부
- 계절차분 포함 모델은 절편 제외가 일반적
4. 모형 추정/진단

---

# 6. 실무 요약
- ETS는 빠르고 해석이 단순하며 baseline으로 강함
- ARIMA는 정상화와 차수 선택이 핵심
- 회귀+ARMA 오차는 외생 변수 반영에 유리
- 최종 채택은 train 지표보다 test 성능과 잔차 진단을 우선
