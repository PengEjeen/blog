# 유로 환율 보고서
201952036 조남영

## 목적
2023-12 ~ 2024-02 유럽 여행 시점을 가정하고, 유로-원 환율 예측으로 가장 유리한 환전 구간을 찾는다.

데이터 출처:
- https://www.kaggle.com/datasets/lsind18/euro-exchange-daily-rates-19992020

원자료는 1999-01-04 ~ 2023-05-26 일 단위 데이터이며, 최근 10년(2014~2023)을 월평균으로 변환해 분석했다.

## 데이터 확인
- 사용 변수: `X.Korean.won..`
- 결측 확인: 없음
- 학습/검증 분할: 최근 10년 월평균 중 앞 8년(train), 뒤 2년(test)

## 모델
비교 모델:
1. ETS
2. ARIMA
3. Regression with ARMA error

---

## ETS
적합 결과 요약:
- 선택 모델: `ETS(M,N,N)`
- 추세/계절성 성분 없이 multiplicative error 중심
- alpha 값이 높아 최근 관측치 반영 강함

진단:
- ACF 상 잔차의 자기상관은 크게 문제되지 않음
- 잔차 분포도 큰 왜도 없이 사용 가능 범위

---

## ARIMA
정상성 점검:
- ACF가 천천히 감소하여 일반 차분 필요
- `ndiffs=1`, `nsdiffs=0`

후보 탐색 및 선택:
- `ARIMA(0,1,2) with drift`
- 진단상 잔차 독립성/정규성은 실무적으로 수용 가능

---

## Regression with ARMA error
- 설명변수로 계절 dummy와 Fourier series를 각각 적용
- 두 방식 중 AICc 기준으로 Fourier 기반 회귀가 우세
- 최종 회귀형은 ARMA 오차를 포함한 시계열 회귀

---

## 성능 비교 (test RMSE)
- ETS: **29.63319**
- ARIMA: 87.06405
- Regression(ARMA error): 152.91986

학습 데이터 기준 성능보다 테스트 성능을 우선해 최종 모델은 ETS로 채택했다.

---

## 2023-12 ~ 2024-02 예측
ETS 예측 결과(요약):
- 2023-12: 가장 낮은 구간으로 추정
- 2024-01, 2024-02: 신뢰구간 포함 기준 12월 대비 불리

결론:
- 3개월 중 **2023년 12월**이 가장 유리한 환전 시점으로 제시됨.

## 부록 (분석 코드 주요 흐름)
```r
library(plyr)
library(fpp2)
library(patchwork)
library(lubridate)
library(zoo)

df <- read.csv("euro-daily-hist_1999_2022.csv")
df_a <- arrange(df, Period.Unit.)
won <- as.numeric(unlist(df_a["X.Korean.won.."]))

start_date <- ymd("2013-05-26")
end_date <- ymd("2023-05-26")
date_range <- seq(start_date, end_date, by = "day")

won_m <- tail(won, 3653)
won_m <- tapply(won_m, ceiling_date(date_range, "month"), mean)
ts.won <- ts(won_m, frequency=12, end=c(2023,05))

train_won <- window(ts.won, end=c(2021,05))
test_won <- window(ts.won, start=c(2021,06))

fit_ets <- ets(train_won)
fit_arima <- auto.arima(train_won, d=1, stepwise=FALSE, approximation=FALSE)

Time <- time(train_won)
res <- numeric(6)
for(i in seq(res)) {
  xreg <- cbind(Time, fourier(train_won, K=i))
  fit <- auto.arima(train_won, xreg=xreg)
  res[i] <- fit$aicc
}

k_min <- which.min(res)
fit_reg <- auto.arima(train_won, xreg=cbind(Time, fourier(train_won, K=k_min)))

fc_ets <- forecast(fit_ets)
fc_arima <- forecast(fit_arima)
fc_reg <- forecast(fit_reg, xreg=cbind(time(test_won), fourier(test_won, K=k_min)))

accuracy(fc_ets, test_won)
accuracy(fc_arima, test_won)
accuracy(fc_reg, test_won)
```
