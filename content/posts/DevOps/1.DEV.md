# DEV (DevOps 인프라 운영 인덱스)

운영 환경에서 자주 다뤘던 인프라 도구별 메모/링크 모음. 항목별 상세는 별도 페이지로 정리.

## 1. 클라우드 / 인프라

- **GCP** — Cloud Run 정적 IP 구성 (VPC Connector + Cloud NAT) → `Cloud/1.GCP.md`
- **Kubernetes** — 클러스터 운영, deployment/ingress 설정
- **SSHFS** — 원격 디스크 마운트로 개발 편의성 확보

## 2. 웹 서버 / 리버스 프록시

- **NGINX** — 리버스 프록시, gzip/캐시, 정적 자원 서빙
- **Apache** — VirtualHost, mod_wsgi, React build 서빙
- **Supervisord with Nginx** — 멀티 프로세스 상시 구동

## 3. CI / 컨테이너

- **Jenkins** → `DevOps/2.Jenkins_Tutorial.md`
- **Docker** — 이미지 빌드, multi-stage build, compose
- **Docker 모니터링 스택** — Prometheus + cAdvisor + Grafana 조합

## 4. 데이터 / 형상 관리

- **MySQL on Ubuntu** — 설치/유저권한/백업
- **Git** — 브랜치 전략, conventional commits

## 5. 자동화 / 사이드 도구

- **Apps Script** — 구글 워크스페이스 자동화 (Sheets/Drive 트리거)
