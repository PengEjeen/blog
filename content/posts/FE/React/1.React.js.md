## 1. 기본 학습 흐름

1. JSX와 컴포넌트 — 함수형 컴포넌트 우선
2. State / Props — 단방향 데이터 흐름
3. Hooks — `useState`, `useEffect`, `useMemo`, `useContext`
4. 라우팅 — `react-router-dom`
5. 빌드 / 배포 — `npm run build` 산출물 정적 호스팅

## 2. 환경 셋업

```bash
# Node 권장 LTS 버전 사용
nvm install --lts
npm create vite@latest my-app -- --template react
cd my-app && npm install && npm run dev
```

## 3. Apache 연동

빌드 산출물(`build/` 또는 `dist/`)을 Apache `DocumentRoot` 로 두고 `mod_rewrite` 로 SPA 라우팅 처리. → 상세는 `FE/React/2.Apache_연동.md`
